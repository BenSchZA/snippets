cadCAD Snippets

A website containing notebooks that showcase features and applications of cadCAD.

If you have any kind of snippet or mini-demo showing how to do something with cadCAD, this is the place to put it! To contribute, fork the repository (Github icon above) and make a pull request containing your snippet additions/changes. Always put your notebooks on the root of the snippets folder and add any dependencies necessary on the requirements.txt file.
